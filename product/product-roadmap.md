# Product Roadmap

## Sections

### 1. Comparator
The home screen and heart of the product — an N-way grid of up to six builds with inline slot editing, focus-mode 1v1 drill-down, and an empty state that funnels mid-game users into common starting points.

### 2. Build Editor
A full-screen deep-dive view of a single build with expanded stat breakdowns, DPS-vs-defense and DPS-over-time graphs, exaltations editing, and per-build scenario overrides — reachable from any comparator column.

### 3. Catalog
A filterable, virtualized browser of every item in the game (weapons, abilities, armors, rings, talismans, sets) with a quick item-vs-item compare mode that does not require committing to a full build.

### 4. Optimizer
A three-mode build search — absolute Best in Slot, best with the user's inventory, and best under user-defined constraints (e.g., min HP/Def, no UTs) — that emits results back into the comparator for further refinement.

### 5. Inventory
An opt-in screen where users mark which items they own (binary toggle) or import their vault and characters from a RealmEye profile, feeding the optimizer's "with my inventory" mode and an optional "owned only" filter elsewhere in the app.
