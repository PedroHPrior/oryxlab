# Reference data sources

Vendored snapshots of community-maintained RotMG asset dumps, used for
cross-validating our `items.json` catalog. These are read-only inputs — never
edit them, just refresh from upstream.

## Sources

### `jrelay-items-2020.xml` (350 KB)
Equipment XML extracted from `ruusey/JRelay`, a 2017–2020-era RotMG packet
sniffer that bundled the official `Items.xml` from the Flash-era client.
Covers ~553 weapons with **full authoritative stats**: `RateOfFire`,
`MinDamage`/`MaxDamage`, `NumProjectiles`, `LifetimeMS`, `Speed`,
`MultiHit`. This is the most reliable independent source for classic
theorycraft items (Crystal Wand, Doom Bow, Demon Blade, Wand of the
Bulwark, etc.).

Caveat: Deca has rebalanced many items between 2020 and now. A weapon's
JRelay numbers and current numbers can legitimately differ.

Source: https://github.com/ruusey/JRelay/blob/master/JRelay/resources/xml/items.xml

### `rotmg-stash-constants-2026.json` (4.6 MB)
Item ID → metadata map extracted from `link3337/rotmg-stash`, a current
Tauri-based stash viewer. Covers **12,119 items** across the entire modern
catalog including PT/event/seasonal items.

Caveat: metadata only — name, slot type, tier, sprite atlas coords, fame
bonus, feed power, bag type, soulbound flag, UT/ST flag. **No DPS-relevant
stats** (damage, RoF, projectile pattern). Useful as a "what items exist"
ground truth, not for stat validation.

Source: https://github.com/link3337/rotmg-stash/blob/main/public/rotmg-assets/constants.json

### `jrelay-drift-report.json`
Generated by `scripts/scrape/validate-against-jrelay.mjs`. Lists every
weapon in our `items.json` whose damage range diverges from JRelay by more
than ±5 absolute points, sorted by percentage delta.

## How to use

```bash
# Re-run validation report against JRelay (no writes):
node scripts/scrape/validate-against-jrelay.mjs

# Apply JRelay numbers as corrections (USE WITH CAUTION — Deca rebalances
# matter, JRelay is 5+ years old):
node scripts/scrape/validate-against-jrelay.mjs --apply
```

## When to update

- After Deca ships a major item rebalance: refresh `rotmg-stash-constants`
  to pick up new items, but DO NOT auto-apply JRelay drift fixes against
  weapons that were intentionally rebuffed.
- A modern (2024+) public XML dump would supersede `jrelay-items-2020.xml`.
  Looking for one is tracked under issue #11.
